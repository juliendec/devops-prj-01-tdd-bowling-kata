Projet DevOps

### Prerequisites

-   [Node JS / NPM](https://nodejs.org/en/)
-   [Yarn (optional)](https://yarnpkg.com/)

### Installing

Install dependencies

```
npm i
```

or

```
yarn
```

## Getting Started

Run the app

```
npm start
```

or

```
yarn start
```
